# Documentation Overview

This folder contains the final M122E documentation material for the Weekly Spotify Wrapped project.

Main files:

- `M122E_Final_Documentation.docx`: main final documentation for submission.
- `M122E_Final_Documentation.md`: complete project documentation.
- `M122E_Test_Protocol.md`: detailed test protocol.
- `M122E_Project_Plan_Spotify_Wrapped.xlsx`: original visual project plan.
- `ProjectProposalM122E.docx`: improved project proposal.
- `../README.md`: installation and usage guide for running the project.

Diagram files:

- `diagrams/system_architecture_diagram.svg`
- `diagrams/uml_activity_diagram.svg`
- `diagrams/uml_class_diagram.svg`
- `diagrams/uml_component_diagram.svg`
- `diagrams/uml_deployment_diagram.svg`
- `diagrams/er_diagram.svg`
- `diagrams/flowchart.svg`

The SVG files can be opened in a browser or inserted into Word/PowerPoint. PNG versions are also included for easier Word/PDF usage, and the main DOCX documentation contains the diagrams as embedded images.
